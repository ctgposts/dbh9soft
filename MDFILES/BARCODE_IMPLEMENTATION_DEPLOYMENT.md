## 🔄 বারকোড ওয়ার্কফ্লো প্রক্রিয়া

```
┌─────────────────────────────────────────────────────────────┐
│                    বারকোড জীবনচক্র                         │
└─────────────────────────────────────────────────────────────┘

PHASE 1: তৈরি (CREATE)
├─ ব্যবহারকারী প্রোডাক্ট ফর্ম পূরণ করে
├─ নাম, ব্র্যান্ড, দাম ইত্যাদি
├─ ফর্ম সাবমিট করে
└─ → Backend Mutation (products.create)

PHASE 2: জেনারেশন (GENERATION)
├─ Timestamp নেওয়া: 123456
├─ ProductCode তৈরি: AB123456
├─ Price Digits: 4500টাকা → 4500 (প্যাডেড)
├─ Barcode তৈরি: AB1234564500
└─ → Uniqueness চেক

PHASE 3: ভ্যালিডেশন (VALIDATION)
├─ বারকোড ইতিমধ্যে আছে কিনা?
├─ হ্যাঁ: ❌ Error "Barcode already exists"
├─ না: ✅ এগিয়ে যান
└─ → ডাটাবেসে সংরক্ষণ

PHASE 4: সংরক্ষণ (STORAGE)
├─ products কালেকশনে ইনসার্ট
├─ barcode ফিল্ডে মূল্য সংরক্ষণ
├─ ইন্ডেক্স: by_barcode আপডেট
└─ → UI তে প্রদর্শনযোগ্য

PHASE 5A: সার্চ - Inventory
├─ ব্যবহারকারী বারকোড টাইপ করে
├─ Real-time সার্চ (Debounce: 300ms)
├─ product.barcode.toLowerCase().includes(searchLower)
├─ ফলাফল ফিল্টার হয়
└─ → স্টক তথ্য প্রদর্শন

PHASE 5B: সার্চ - POS
├─ বারকোড স্ক্যানার স্ক্যান করে
├─ POS সার্চ বারে স্বয়ংক্রিয় ইনপুট
├─ p.barcode?.includes(filters.searchTerm)
├─ রিলেভেন্স স্কোরিং: 3 পয়েন্ট
└─ → প্রোডাক্ট তাত্ক্ষণিক প্রদর্শিত

PHASE 6: প্রিন্টিং (PRINTING)
├─ Barcode Manager খুলুন
├─ প্রোডাক্ট সিলেক্ট করুন
├─ স্তিকার সেটিংস কনফিগার করুন
├─ প্রিন্ট করুন (বারকোড ভিজ্যুয়াল + নাম্বার)
└─ → ফিজিকাল স্টিকার প্রাপ্ত

PHASE 7: ব্যবহার (USAGE)
├─ স্টিকার প্রোডাক্টে লাগান
├─ বিক্রয়ের জন্য প্রস্তুত
└─ → বারকোড স্ক্যানার দিয়ে বিক্রয় সম্পূর্ণ

PHASE 8: ট্র্যাকিং (TRACKING)
├─ প্রতিটি বিক্রয় রেকর্ড হয়
├─ বারকোড দ্বারা সনাক্তকৃত
└─ → শেষ পণ্যের ট্র্যাকিং সম্পূর্ণ
```

## 🎯 ব্যবহারকারী গাইড

### 1️⃣ প্রোডাক্ট তৈরি (Product Creation)

```
ধাপ 1: Inventory → "Add Product" ক্লিক করুন
ধাপ 2: ফর্ম পূরণ করুন:
        - নাম: প্রোডাক্টের নাম
        - ব্র্যান্ড: DBH
        - ক্যাটেগরি: বেছে নিন
        - ফ্যাব্রিক: বেছে নিন
        - কালার: বেছে নিন
        - সাইজ: বেছে নিন
        - দাম: টাকায় প্রবেশ করুন
        - স্টক: পরিমাণ দিন
        *(বারকোড ফিল্ড খালি রাখুন - স্বয়ংক্রিয় হবে)*
ধাপ 3: "Create Product" ক্লিক করুন
ধাপ 4: ✅ প্রোডাক্ট সফলভাবে তৈরি হয়েছে
        বারকোড: AB[XXXXXX][DDDD] (স্বয়ংক্রিয়ভাবে তৈরি)
```

**নোট**: কাস্টম বারকোড দিতে চাইলে বারকোড ফিল্ড পূরণ করুন, কিন্তু অবশ্যই অনন্য হতে হবে।

### 2️⃣ Inventory প্রোডাক্ট খোঁজা (Finding in Inventory)

```
ধাপ 1: Inventory পেজ খুলুন
ধাপ 2: সার্চ বারে বারকোড টাইপ করুন
        উদাহরণ: "AB1234564500"
        অথবা আংশিক: "AB1234"
ধাপ 3: স্বয়ংক্রিয়ভাবে ফিল্টার হবে
ধাপ 4: প্রোডাক্ট সারি ক্লিক করে বিস্তারিত দেখুন

বিকল্প সার্চ শব্দ:
  - পূর্ণ বারকোড: "AB1234564500"
  - আংশিক বারকোড: "AB12"
  - প্রোডাক্ট কোড: "AB123456"
  - প্রোডাক্ট নাম: "Abaya"
  - ব্র্যান্ড: "DBH"
```

### 3️⃣ POS এ প্রোডাক্ট খোঁজা (Finding in POS with Scanner)

#### বিকল্প A: বারকোড স্ক্যানার ব্যবহার করা (Recommended)

```
ধাপ 1: Enhanced POS পেজ খুলুন
ধাপ 2: বারকোড স্ক্যানার সংযুক্ত করুন (USB/Bluetooth)
ধাপ 3: সার্চ বারে ক্লিক করুন (সবুজ indicator)
ধাপ 4: স্ক্যানার দিয়ে বারকোড স্ক্যান করুন
        → স্বয়ংক্রিয়ভাবে সার্চ হবে এবং প্রোডাক্ট দেখা যাবে
ধাপ 5: প্রোডাক্ট ক্লিক করুন বা তার উপর ডাবল ক্লিক করুন
ধাপ 6: সাইজ নির্বাচন করুন (প্রয়োজনে)
ধাপ 7: কার্টে যোগ হবে
ধাপ 8: পরিমাণ সমন্বয় করুন
ধাপ 9: কেনাকাটা সম্পূর্ণ করুন
```

#### বিকল্প B: ম্যানুয়াল সার্চ (যখন স্ক্যানার নেই)

```
ধাপ 1: Enhanced POS পেজ খুলুন
ধাপ 2: সার্চ বারে বারকোড টাইপ করুন
        "AB1234564500"
ধাপ 3: Enter চাপুন অথবা অপেক্ষা করুন
ধাপ 4: প্রোডাক্ট ফিল্টার হবে
ধাপ 5: প্রোডাক্ট কার্টে যোগ করুন
ধাপ 6: বাকি প্রক্রিয়া একই
```

### 4️⃣ বারকোড স্টিকার প্রিন্ট করা (Printing Barcode Stickers)

```
ধাপ 1: Inventory খুলুন
ধাপ 2: "Barcode Manager" ট্যাব ক্লিক করুন (আইকন: 📊)
ধাপ 3: প্রোডাক্ট সার্চ করুন বা তালিকা থেকে বেছে নিন
ধাপ 4: প্রোডাক্টের পাশে চেকবক্স চেক করুন
        (এক বা একাধিক প্রোডাক্ট নির্বাচন করতে পারেন)
ধাপ 5: স্টিকার সেটিংস কনফিগার করুন:
        - স্টিকার আকার: ডিফল্ট 2" × 1.5"
        - কাগজ আকার: ডিফল্ট 8.5" × 11"
        - অপশন: নাম, দাম, সিরিয়াল নাম্বার অন্তর্ভুক্ত করুন/না করুন
ধাপ 6: "Generate & Print" ক্লিক করুন
ধাপ 7: প্রিভিউ উইন্ডো খুলবে - স্টিকার দেখুন:
        □ স্টোর নাম: "DUBAI BORKA HOUSE"
        □ প্রোডাক্ট নাম
        □ দাম
        □ বারকোড (ভিজ্যুয়াল সব্জিত লাইন)
        □ বারকোড নাম্বার (নীচে পঠনযোগ্য): AB1234564500 ✅
        □ অবস্থান (BOX-5)
        □ সাইজ/কালার
ধাপ 8: "Print Labels" বাটন ক্লিক করুন
ধাপ 9: প্রিন্টার সিলেক্ট করুন
ধাপ 10: প্রিন্ট করুন এবং প্রসেস সম্পূর্ণ করুন
```

**স্টিকার ডিজাইন বৈশিষ্ট্য**:
- ✅ পেশাদার রঙ স্কিম (সাদা ব্যাকগ্রাউন্ড)
- ✅ পঠনযোগ্য ফন্ট
- ✅ বারকোড ভিজ্যুয়াল এবং টেক্সট উভয়ই
- ✅ সমস্ত গুরুত্বপূর্ণ পণ্য তথ্য
- ✅ পুনরাবৃত্তিযোগ্য সাদা স্থান সীমা

- বারকোড ফরম্যাট: AB[Timestamp][Price]
- ইউনিকনেস গুরুত্ব

**Module 2: Inventory সার্চ (10 মিনিট)**
- সার্চ বার খোলা
- বারকোড টাইপ বা স্ক্যান করা
- ফলাফল পড়া এবং ক্রিয়া গ্রহণ

**Module 3: POS স্ক্যানিং (15 মিনিট)**
- স্ক্যানার সংযোগ এবং পরীক্ষা
- স্ক্যান এবং কার্ট যোগ করা
- বিক্রয় প্রক্রিয়া সম্পূর্ণ করা

**Module 4: স্টিকার প্রিন্টিং (20 মিনিট)**
- Barcode Manager খোলা
- প্রোডাক্ট নির্বাচন করা
- সেটিংস কনফিগার করা
- প্রিন্ট করা এবং লেবেলিং করা

**মোট প্রশিক্ষণ সময়**: 60 মিনিট
